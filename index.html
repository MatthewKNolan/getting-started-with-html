The World Wide Web is made up of HTML and Hyperlinks.

There are four major components of accessing any webpage. These are:
1.) You
2.) Your computer and your browser
3.) The Internet
4,) Servers
*Servers are computers optimized to sit in a data center and store files.

HTML Markup
-Consists of an opening tag, a closing tag, and contents.
Examples:
< b > Hello < / b > will bold the word Hello
< em > Hi < / em > will italicize or place emphasis on the word Hi

HTML Attributes
Attributes are assigned using the following two tags:
< tag attr = "value" > CONTENTS < / tag>
< a href="value"> CONTENTS < /a > will add a hyperlink to word in the CONTENTS.
< img src="url" alt="text"> will add an image from the source URL.
-Adding text to ALT will display alternate text when no image is found.
-Image is a void text, meaning there is no actual content.

Inline and Block
-Inline ends line and appears as one tag at the break like < br >.
-Block will create a box around the content which has height and width.
This has both an opening and closing tag. An example is < p > and < /p >.
